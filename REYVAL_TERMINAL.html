The Core Matrix: Technology vs Library

Technology (Stack)

Node.js (v20+)Runtime de ejecución de alto rendimiento asíncrono.
NestJS (v11)Framework de backend con arquitectura modular basada en DI (Dependency Injection).
Angular (v21)Framework de cliente basado en Signals para una gestión reactiva pura.

Backend Libraries

Prisma v5.2Toolkit de persistencia Type-Safe con introspección de esquemas.

Operational Manual

Protocolos de Emergencia y Gestión de Infraestructura

Gestión de RAM (Regla de Oro)

El servidor de Oracle Cloud tiene recursos limitados (~1GB RAM). El exceso de procesos Java puede saturar el sistema.

Máximo 2 Apps activas a la vez.

Comandos SSH de Emergencia

Reiniciar Nginx:sudo systemctl restart nginx
Ver Logs del Sistema:journalctl -u reyval-backend --since "1 hour ago"

Protocolo de Despliegue (Build -> Transport -> Restart)

  1. Generar el paquete atómico en local usando el botón BUILD BACKEND.
  2. Verificar que el log termine con "Build Success".
  3. Presionar DEPLOY para subir vía túnel SCP y reiniciar el servicio Systemd remoto.
