2. Descripción: Importación Masiva de Activos

ObjetivoCargar cientos de registros (lotes, usuarios, pagos) de forma atómica y segura.
ActoresAdministrador (Iniciador), Sistema (Pasivo).
IniciaciónEl Administrador sube un archivo .xlsx o .csv en la sección de "Herramientas".
Flujo de Mensajes 1. El Administrador selecciona el tipo de entidad a cargar.
2. El Administrador sube el archivo al servidor.
3. El Sistema parsea el contenido y valida la estructura de columnas.
4. El Sistema ejecuta validaciones cruzadas (ej. ¿Existe el fraccionamiento?).
5. El Sistema presenta un resumen de éxitos/fallos.
6. El Administrador confirma la persistencia final.
Flujos Alternativos A1: Error de Formato: El Sistema detiene la carga e indica la fila y columna con error.
A2: Datos duplicados: El Sistema permite ignorar o actualizar registros existentes.
Valor FinalBase de datos actualizada con integridad garantizada en segundos.
