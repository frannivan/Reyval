2. Descripción: Simulación de Crédito

ObjetivoProyectar el escenario financiero de adquisición para un cliente potencial.
ActoresVendedor/Cliente (Iniciadores), Sistema (Pasivo).
IniciaciónSe selecciona un lote disponible y se accede al botón "Simular".
Flujo de Mensajes 1. El Usuario ingresa el monto del enganche y plazo (meses).
2. El Sistema valida los límites permitidos para ese desarrollo.
3. El Sistema calcula el interés (si aplica) y la cuota mensual.
4. El Sistema genera la tabla de amortización detallada.
5. El Usuario solicita exportar la cotización oficial.
Flujos Alternativos A1: Enganche fuera de rango: El Sistema sugiere el monto mínimo requerido.
A2: Plazo no permitido: El Sistema bloquea la simulación hasta ajustar el plazo.
Valor FinalUn documento de tabla de amortización que sirve como base para el contrato legal.
