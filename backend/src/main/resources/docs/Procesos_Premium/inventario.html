2. Descripción: Configurar Desarrollo e Inventario

ObjetivoDigitalizar y parametrizar un nuevo fraccionamiento para su comercialización.
ActoresAdministrador (Iniciador), Sistema (Pasivo).
IniciaciónEl Administrador accede al panel de "Desarrollos" y selecciona "Crear Nuevo".
Flujo de Mensajes 1. El Administrador ingresa datos generales (Nombre, Ubicación, Logo).
2. El Administrador carga el archivo SVG del plano maestro.
3. El Sistema procesa el SVG para identificar polígonos (path ID).
4. El Administrador vincula cada polígono con un número de manzana y lote.
5. El Sistema guarda la configuración y habilita el plano para consulta pública.
Flujos Alternativos A1: Error en SVG: El Sistema notifica que el archivo no tiene IDs válidos.
A2: Lote ya existente: El Sistema previene la duplicidad de coordenadas.
Valor FinalUn mapa interactivo funcional donde cada píxel representa una unidad de negocio con valor real.
