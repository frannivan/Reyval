2. Descripción: Generación de Contrato

ObjetivoGenerar el instrumento jurídico vinculante con los datos del adquirente y el lote.
ActoresVendedor (Iniciador), Sistema (Pasivo).
IniciaciónEl Vendedor confirma que el expediente está completo y selecciona "Generar Contrato".
Flujo de Mensajes 1. El Vendedor solicita la creación del PDF.
2. El Sistema recupera los datos del cliente, el lote y la simulación financiera.
3. El Sistema inyecta los datos en la plantilla jurídica vigente.
4. El Sistema convierte los montos de número a letra (validación legal).
5. El Sistema devuelve el archivo PDF listo para su descarga e impresión.
Flujos Alternativos A1: Expediente Incompleto: El Sistema bloquea la generación hasta subir INE o firma.
Valor FinalUn archivo PDF/A inalterable que documenta la venta formalmente.
